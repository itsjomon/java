## 3. Find maximum of 3 numbers.
**Pseudocode**
1. Start
2. Input a, b & c
3. ```
   if a>b do
      if a>c
         print a
      else
         print c
   else
     if b>c do
        print b
     else
        print c
   ```
4. Exit

## 4.Find if a number is Prime or not.
**Pseudocode**
1. Start
2. Input n 
3. let div=2
4. ```
   while div<n do
   if n%div == 0 do
      print not prime
      exit
   else
     div = div+1
   ```
5. Print prime
6. Exit
